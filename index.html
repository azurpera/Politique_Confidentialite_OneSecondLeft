Politique de Confidentialité

Dernière mise à jour : 29 Mars 2026

Bienvenue sur One Second Left. La protection de vos données privées est notre priorité absolue. Ce document explique quelles données nous collectons, comment nous les utilisons, et comment vous pouvez contrôler ces informations.

1. Données Collectées

L'application utilise les services de Google (Firebase et AdMob). Selon votre mode d'utilisation, certaines données peuvent être collectées :

A. Si vous jouez en Mode Invité (Anonyme)

  • Identifiant unique (UID) : Un identifiant Firebase généré aléatoirement pour sauvegarder votre meilleur score localement de façon anonyme.
  • Identifiants publicitaires : Google AdMob peut collecter des identifiants d'appareils et des données d'interaction pour vous afficher des publicités (soumises aux paramètres de non-personnalisation pour respecter les directives familiales).

B. Si vous jouez avec Google Sign-In

  • Adresse E-mail : Utilisée exclusivement pour authentifier votre compte via Firebase Auth. Elle n'est jamais affichée publiquement.
  • Nom d'affichage et Photo de profil : Récupérés depuis votre compte Google pour être affichés dans le classement mondial (Leaderboard).
  • Données de Jeu : Votre score maximum, votre niveau atteint et votre nombre de clics exacts, sauvegardés sur nos serveurs Firestore sécurisés pour le classement.

2. Utilisation de vos Données

Vos données ne sont utilisées que pour le fonctionnement propre de l'application :

  • Vérifier votre identité (Firebase Auth).
  • Sauvegarder votre progression et la synchroniser sur vos appareils.
  • Alimenter le classement mondial des joueurs.
  • Fournir un affichage publicitaire permettant de maintenir le jeu gratuit.

3. Partage d'Informations

Nous ne vendons ni ne louons jamais vos informations personnelles à des tiers. Les seules entités avec lesquelles les données sont partagées sont :

  • Google Cloud (Firebase / Firestore) : Hébergement de la base de données et gestion d'authentification.
  • Google AdMob : Réseau publicitaire (avec conformité Child-Directed / non-personnalisé selon le paramétrage standard G-rating).

4. Droit de Suppression et de Contrôle

Vous êtes maître de vos données. L'application intègre une option directe de suppression de vos données :

  • Allez dans les paramètres de l'application en cliquant sur l'icône d'engrenage.
  • Descendez et cliquez sur le bouton rouge "SUPPRIMER MON COMPTE ET MES DONNÉES".
  • Cette action est immédiate et irréversible : l'intégralité de vos scores Firestore et de votre profil Firebase Auth seront instantanément et définitivement effacés.

5. Sécurité

Nous utilisons l'infrastructure ultra-sécurisée de Google Firebase. Toutes les transmissions de données (authentification, intégration des scores) sont chiffrées via les protocoles TLS/HTTPS.

6. Contact

Si vous avez des questions ou des inquiétudes concernant cette politique de confidentialité, vous pouvez nous contacter à l'adresse e-mail de support définie sur notre fiche Google Play Store.
